The first 60 seconds

Four taps and Clicks is your home screen.

A real out-of-box flow — no configuration marathon. Each step is a single choice, and you can change any of them later just by typing.

1

Set Clicks as your home app

Android asks which launcher to use — pick Clicks and press Always. Your old launcher stays installed; the home button just comes here now.

Reversible anytime in Android settings
2

Choose your keyboard style

Clicks' one first-run question. It's how the whole launcher feels, so you pick up front — and switch whenever you want.

⌨️ Docked default

The keyboard stays fixed at the bottom. Start typing and search takes over the screen — the command-console feel.

🧩 Widget

The keyboard sits on the home screen as a compact instrument. A DOCK key snaps back to the fixed layout.

3

Turn the keyboard on

Enable the Clicks keyboard once in the system dialog and set it as your input method. Now the same engine powers both search and everyday typing.

On-device prediction & glide typing, in 6 languages
4

Grant only what you use

Nothing is demanded up front. Clicks asks for a permission the moment a feature needs it — weather wants location, the people widget wants notification access — and explains why, right there.

Skip any of them; the rest of the launcher still works
