The tall-phone problem nobody fixed

Field notes · 4 min read

Phones kept getting taller, but the search bar stayed pinned to the top. Every lookup starts with a reach — thumb straining to the far corner just to raise the keyboard.

What if the keyboard came to the thumb instead? A handle at the bottom you pull up when you need it, that drops away the moment you scroll.

Scroll this page while the keyboard is up — watch it tuck itself away, then pull it back with one motion.

The address bar above focuses on its own as the keyboard rises, so you can start typing without ever touching it.
